Hibernate Reactive is a reactive API for Hibernate ORM, supporting non-blocking database drivers and a reactive style of interaction with the database.

The central APIs are defined as inner interfaces of

  • {@link org.hibernate.reactive.mutiny.Mutiny}, for programs using Mutiny reactive streams, and
  • {@link org.hibernate.reactive.stage.Stage}, for programs using Java's {@link java.util.concurrent.CompletionStage}.
An instance of {@link org.hibernate.reactive.mutiny.Mutiny.SessionFactory} or {@link org.hibernate.reactive.stage.Stage.SessionFactory} may be obtained by calling {@link javax.persistence.EntityManagerFactory#unwrap(Class)}.

The public API is organized into the following packages:

{@link org.hibernate.reactive.mutiny}— APIs specific to usage with Mutiny reactive streams
{@link org.hibernate.reactive.stage}— APIs specific to usage with {@link java.util.concurrent.CompletionStage}
{@link org.hibernate.reactive.common}— APIs common to both kinds of reactive stream
{@link org.hibernate.reactive.id}— SPI for plugin custom identifier generation
{@link org.hibernate.reactive.pool}— SPI for plugin custom connection pooling
{@link org.hibernate.reactive.vertx}— SPI for plugin custom integration with Vert.x

SPIs are implemented as Hibernate services.

Packages not listed above are considered non-public and are subject to change without notice.
